# Conditionals in Shell Scripting

## If Statements
- if [ condition ]; then ... fi  
- if [ condition ]; then ... else ... fi  
- if [ condition1 ]; then ... elif [ condition2 ]; then ... else ... fi  

## Test Operators
- -eq → Equal (numbers)  
- -ne → Not equal (numbers)  
- -lt → Less than  
- -le → Less than or equal  
- -gt → Greater than  
- -ge → Greater than or equal  
- =  → String equal  
- != → String not equal  
- -z → String is empty  
- -n → String is not empty  
- -f → File exists and is regular  
- -d → Directory exists
