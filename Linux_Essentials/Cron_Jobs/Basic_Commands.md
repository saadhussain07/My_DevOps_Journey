
## Commands
- crontab -e → Edit crontab  
- crontab -l → List jobs  
- crontab -r → Remove crontab  

# Managing Cron Jobs

## View Current Jobs
- crontab -l  

## Edit Cron Table
- crontab -e  

## Remove All Jobs
- crontab -r  

## Logs
- /var/log/cron (RHEL/CentOS)  
- /var/log/syslog (Ubuntu/Debian)
